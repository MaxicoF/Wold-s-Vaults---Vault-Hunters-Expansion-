> Для связи используйте возможности github или перейдите в профиль и воспользуйтесь ссылкой на Telegram.

# О переводе
wold's vaults translate to russian language
Этот перевод создан для сборки Wold's Vaults - Vault Hunters Expansion (https://www.curseforge.com/minecraft/modpacks/wolds-vaults) 
Он является частично машинным + с использованием ИИ, частично выполнен вручную.
Некоторые части перевода (в основном квесты) взяты из репозитория:
https://github.com/Linaryx/wold-s-vaults-translate-to-russian/tree/main — с последующими правками и адаптацией.

> Перевод будет постепенно улучшаться в процессе игры: некоторые формулировки могут быть уточнены, а ошибки — исправлены.

> Я открыт для предложений и готов вносить правки, если у вас есть замечания или улучшения для перевода.

# Установка
1. Скачайте архив с переводом.
1. Распакуйте его.
1. Скопируйте файлы с заменой в папку сборки.

# !!!! **Для исправление кодировки** 
Чтобы русский текст отображался корректно, необходимо добавить в параметры Java следующую строку:
```
-Dfile.encoding=UTF-8
```
